Usage
=====

1. Install the module in your Odoo instance.

2. Add field in your model to include the ``attachment_ids`` field.

.. code:: python

   attachment_ids = fields.Many2Many('ir.attachment', string='Attachments')

3. In your xml file, include the ``many2many_binary`` widget:

.. code:: xml

   <field name="attachment_ids" widget="many2many_binary"/>

4. The PDF preview will be displayed in the form view of the model where
   you added the ``attachment_ids`` field.
